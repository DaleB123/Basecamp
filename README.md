# Basecamp

For college students who plan group trips who are frustrated by the disorganization, fragmented communication, and a lack of clear decision-making, Basecamp is a collaborative trip planning application that centralizes all communication, simplifies itinerary building, and enables easy group decision-making through an intuitive interface. Unlike scattered group chats, email threads, and manual spreadsheets, our product provides a single, organized platform where every member can contribute to the plan, track costs, and vote on activities, making group travel planning a seamless and enjoyable process.

## How to set up environment

Clone the repository and be on the "dev" branch.

Ensure you have Docker Desktop installed.
Navigate to the scripts folder in the terminal using ```cd scripts``` and run the start-dev script with ```./start-dev```

Once the containers are built and running, in your browser go to http://localhost:3000
